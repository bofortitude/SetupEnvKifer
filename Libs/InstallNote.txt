1. Before installing the pycrypto, please install the gcc for your OS first.
Example on Ubuntu:
# apt-get install -y gcc

2. Install python-dev for installing pycrypto
# apt-get install -y python-dev

3. Install pycrypto
# ./configure
# python setup.py install

4. Install requirement components:
# apt-get install -y libffi-dev
# apt-get install -y libssl-dev

4. Install paramiko
# python setup.py install
